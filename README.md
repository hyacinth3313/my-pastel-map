# 🌸 Soft Pastel Map

An interactive, aesthetic city guide and personal world explorer. This project uses **Leaflet.js** and **OpenStreetMap** (via CartoDB) to create a soft, Bridgerton-inspired map experience.

## ✨ Features
* **Custom Markers**: Add and categorize your favorite spots (Cafés, Libraries, Parks, Art).
* **Permanent Memory**: Your saved spots are remembered by your browser even after refreshing.
* **Smart Routing**: Click any marker and select "Route Here" to see a pastel pink path from your starting pin.
* **English Labels**: Uses a clean, English-first map layer for global readability.
* **Custom Typography**: Features the elegant *Bridgerton* font for a sophisticated feel.

## 🚀 How to Use
1. **Explore**: Use the search bar at the top right to find specific addresses.
2. **Add a Spot**: Click anywhere on the map to name and categorize a new location.
3. **Get Directions**: 
   * Click "Show Directions" in the sidebar to reveal the routing pins.
   * Drag the **Green pin** to your start and the **Red pin** to your end.
   * Or, click a marker's "Route Here" button to snap the destination instantly.
